# 🌙 GlowMoon

> A minimalist period tracking app built with SwiftUI.

## 📌 Project Roadmap
- [ ] **Phase 1: Architecture** - Setup project structure and theme.
- [ ] **Phase 2: Onboarding** - User introduction and initial data setup.
- [ ] **Phase 3: Home & Calendar** - The main tracking interface.
- [ ] **Phase 4: Data Persistence** - Integrating SwiftData.
- [ ] **Phase 5: Insights** - Cycle history and predictions.

## 🛠 Tech Stack
- **Language:** Swift 5.10+
- **Framework:** SwiftUI
- **Database:** SwiftData
- **Minimum iOS:** 17.0
